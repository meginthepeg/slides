What is GIS?

Geographic Information System

Any system used to analyze, capture or work with geospatial data.

which is basically jargon for “data/information which you would display with a map

1. What data types are used in GIS


Data Types: Tabular

table or text file

Data Types: Vector (point, line, polygon)


points, lines and areas. each point has an x,y (lat/long) coordinate pair associated with it

Data Types: Raster

A grid made of cells/ pixels(.jpg, .tiff, .grid, .rast…)
